A next path group defines a switch for the current path group when the character terminates a path in the current path group. When the character switches over to another path group, it can follow an optional transitioning path that belongs to the current path group to connect to the next one's rendezvous.

Right click the node to move up, move down or remove the next path group from the path group. You can also change the name of the next path group along with an optional transitioning path in the below input boxes.
