Define the group navigation mode when moving toward the waypoint, taking one of the values 1, 2 and 3.

  1. Everyone moves along the current path synchronously, with each waypoint actions executed at the same time.
  2. Only the master moves along the current path without waiting for other slaves, who are always following the master by moving toward the same waypoint as the master.
  3. Only the master moves along the current path without waiting for other slaves, who are always following the master unit.
