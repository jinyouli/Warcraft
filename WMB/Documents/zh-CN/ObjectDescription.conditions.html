This is a group of condition expressions, evaluated by by combining them altogether to a boolean value. The type of the conditions can be either "and" or "or", which indicates either all of them or one of them evaluates to true, the altogether result is true.

The conditions reside under various nodes. They can vary as both pre-conditions and finish conditions. Make sure you are clear about it before you actually start to define the conditions.

Right click the node to add a new condition, with the following condition expression syntax.

VARIABLE OPERATOR CONSTANT

For example,

myself.level > 5
